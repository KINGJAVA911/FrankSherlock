# v0.3.0

### PDF classification
- PDFs now go through the vision model for classification, same as images. Previously only OCR text was extracted.

### Ollama management
- Switched from shelling out to `ollama` CLI to using the Ollama HTTP API directly. More reliable detection of running instances and model status.

### UI
- Properties modal showing full file metadata (path, size, dates, classification details)
- "Copy Path" option for folders in the sidebar context menu
- Escape key now closes any open modal
- Preview collage limit raised from 4 to 10 images, with a responsive grid layout
- Scan "Cancel" button renamed to "Pause" (since scans resume from where they left off)

### Scanning
- Images under 10 KB are skipped during scanning to filter out web icons, spacers, and favicons

### Platform
- macOS builds are now ad-hoc signed with entitlements for the hardened runtime
- Fixed cross-platform path handling issues (Windows UNC paths, OS-specific imports)
- Consolidated all OS-specific code into the `platform/` module
