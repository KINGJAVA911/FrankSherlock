# v0.4.0

### Duplicate detection
- Find exact duplicates by SHA-256 fingerprint. The "keeper" heuristic picks the oldest file with the shortest path.
- Find near-duplicates using dHash for visual similarity and Jaccard word overlap on descriptions (85/15 weighting). Threshold is adjustable.
- Compare all files in a duplicate group side by side (up to 10), with numbered labels and file metadata.
- Duplicate groups are color-coded: green for exact matches, yellow for near-duplicates (≥85% similar), red for lower-confidence matches that need a visual check.

### PDF password manager
- Save passwords to a global pool. They're tried automatically against protected PDFs during scans and when previewing.
- In the PDF viewer, saved passwords are tried first. If you type a new password and it works, it's added to the pool.
- Files that get unlocked are re-classified and thumbnailed automatically.
- "Retry All" button in the PDF Passwords panel runs all saved passwords against every still-protected PDF.

### Scan performance
- Discovery phase is faster: directory subtrees are pruned early, DB writes are batched (200 per transaction), and progress is checkpointed every 50 files.
- New progress indicator during discovery shows "Discovering files... (N found)" with an animated bar.
- You can now cancel a scan during the discovery phase, not just during classification.

### macOS
- The DMG is now signed with a Developer ID certificate and notarized with Apple. No more right-click-to-open workaround.

### Linux
- The AppImage now ships with a `.asc` GPG signature for download verification.

### Bug fixes
- Clicking a root, album, or smart folder in the sidebar now properly exits duplicates mode.
- Password-protected PDFs skip cleanly instead of failing multiple times.
- Root folders that share a name are disambiguated in the sidebar (e.g. "..user/Pictures" vs "..drive/Pictures").
