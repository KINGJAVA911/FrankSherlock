# v0.2.0

### Search
- FTS now uses AND semantics by default (all words must match)
- New `subdir:` filter to search within a specific subdirectory
- Root path overlaps are detected and handled when adding new roots

### Scan UX
- Scan progress is now shown directly in the sidebar RootCard instead of a separate panel
- Drag-and-drop reorder for root folders in the sidebar
- Cancel is more responsive: the flag is checked after each classification, and the UI polls immediately
- Scans are cancelled automatically before deleting a root

### Bug fixes
- Fixed scan cancel sometimes taking a long time to respond
