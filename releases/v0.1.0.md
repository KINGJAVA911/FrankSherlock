# v0.1.0 — Initial release

First public release of Frank Sherlock.

### What's in the box
- Tauri v2 desktop app (Linux AppImage, macOS DMG, Windows MSI)
- Ollama-powered vision classification (qwen2.5vl) with 3-tier JSON fallback parsing
- Incremental scanning: unchanged files (same mtime + size) are skipped entirely
- Move detection: renamed/moved files keep their classification data
- Thumbnails: 300px JPEG with Lanczos3 resampling, GIF first-frame support
- SQLite + FTS5 full-text search with natural language query parsing (media type, dates, confidence filters)
- PDF scanning, indexing, and in-app preview via PDFium
- Surya OCR in an isolated Python venv, auto-provisioned from the setup modal
- Albums and smart folders (saved queries)
- Multi-select with Shift/Ctrl click, collage preview, clipboard copy
- Sidebar with drag-and-drop root reorder, scan queue, cancel/resume
- Context menu: copy, delete, rename, copy description/OCR text
- Sort controls (name, date, type, relevance)
- F1 help dialog with query syntax examples
- Hardware-aware model selection (3B/7B/32B based on GPU/RAM)
- CLI folder argument support for opening or adding roots from the terminal
- Auto-cleanup of orphaned thumbnails and classification cache
- CI on Linux, macOS, and Windows; release workflow with GitHub Actions
