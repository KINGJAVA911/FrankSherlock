# v0.7.0

### Face detection and recognition
- Detect faces in your photo library using native ONNX models (SCRFD for detection, ArcFace for recognition) -- no Python or external services needed.
- Models are downloaded automatically on first use from the InsightFace buffalo_l pack.
- 512-dimensional face embeddings enable identity-aware clustering with cosine similarity (threshold 0.45).
- Faces are automatically clustered into people after detection. New faces are assigned to existing clusters or start new ones.
- FacesView shows all detected people as a card grid with representative face crops, editable names, and face count badges.
- Name people, merge duplicate identities, and remove false detections from the FacesView UI.
- Face crops are saved as 150px JPEGs in the cache directory.
- Search by person: `face:alice`, `face:"Full Name"`, or `face:42` (by person ID).
- Face detection is triggered per-folder from the right-click context menu.
- Status bar shows face detection progress with download/loading/detecting phases.
- Re-cluster confirmation guard prevents accidental overwrites of manual name assignments.

### Preview keyboard navigation
- Press Space to close the preview modal (previously Space only opened it).
- ArrowLeft/ArrowRight navigate to previous/next image while preview is open (single-item mode).
- Arrow keys are ignored in multi-image collage/comparison mode.
- Keyboard events skip handling when a video element or input field has focus.

### Setup modal install links
- When Ollama is not detected, the setup modal now shows a clickable "install" link that opens the Ollama download page in the system browser.
- When Python is not available for OCR, a clickable "install Python" link opens the Python downloads page.
- Uses the official `tauri-plugin-opener` for cross-platform URL opening.

### Internal improvements
- App.tsx refactored to extract 5 feature hooks (`useAutoUpdate`, `useFaceDetection`, `useAlbumManager`, `useSmartFolderManager`, `useDuplicatesManager`), reducing orchestration complexity.
- Face feature code extracted into shared helpers and CSS modules.
- Thumbnail generation now respects EXIF orientation data.
- Database migrations 11-12 add face detection tables (`face_detections`, `people`) with proper indexes.
- Test suite expanded to 322 Rust tests and 299 frontend tests.
